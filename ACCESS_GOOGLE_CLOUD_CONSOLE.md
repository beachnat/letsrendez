# How to Access Google Cloud Console

## Quick Access - Direct Links

### Option 1: Direct Link to OAuth Consent Screen
Click this link (it should take you right there):
```
https://console.cloud.google.com/apis/credentials/consent?project=letsrendez
```

### Option 2: Google Cloud Console Home
```
https://console.cloud.google.com
```
Then select your **letsrendez** project from the dropdown at the top.

## Step-by-Step Navigation

### From Firebase Console (Easiest!)

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Select your **letsrendez** project
3. Click the **gear icon** (⚙️) → **Project Settings**
4. Scroll down to find **Project Settings** section
5. Look for a link that says:
   - "Open in Google Cloud Console" or
   - "Project settings" with a link to Cloud Console
6. Click that link - it will take you to Google Cloud Console

### Direct Navigation

1. Go to: https://console.cloud.google.com
2. **Important:** Make sure you're logged in with the SAME Google account you used for Firebase
3. At the top, you'll see a project dropdown
4. Click it and select **letsrendez** (or search for it)
5. If you don't see it, click "Select a project" and search for "letsrendez"

### Once in Google Cloud Console

1. Look at the **left sidebar menu** (hamburger menu ☰ if collapsed)
2. Go to **APIs & Services** → **OAuth consent screen**
3. Or use the search bar at the top and type "OAuth consent"

## What If You Can't Find It?

### Check Your Account
- Make sure you're logged into the **same Google account** you used for Firebase
- Firebase and Google Cloud Console use the same account

### Check Project Access
- Make sure you have access to the letsrendez project
- If you created the Firebase project, you should have access

### Alternative: From Firebase
The easiest way is from Firebase Console:
1. Firebase Console → Project Settings
2. Look for "Google Cloud Console" link
3. Click it - takes you directly there
